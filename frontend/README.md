# lung-nodule-frontend

## Start

```bash
npm install
npm run dev
```

- Dev URL: http://localhost:5173
- API proxy: /api -> http://localhost:8080

## Pages

- Public: `/login`, `/register`
- Workspace: `/app/workspace`
- Patient: profile, studies, detail, registration, upload
- Doctor: profile, patients, studies, detail, reports, annotations
- Admin: dashboard, users
